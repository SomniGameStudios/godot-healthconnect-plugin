# Godot HealthConnect Plugin

A Godot 4 Android plugin that provides native Google Health Connect step counting and activity tracking for Android games.

[![GitHub stars](https://img.shields.io/github/stars/SomniGameStudios/godot-healthconnect-plugin.svg?style=social&label=Star)](https://github.com/SomniGameStudios/godot-healthconnect-plugin)

## Features

- **Health Connect** — Query today's steps and activity data.
- **Auto-Injection** — Required permissions are automatically handled in your Android export.
- **Easy Integration** — Clean GDScript wrapper for Android-specific logic.

## Quick Start (Pre-built)

1. Download the latest release zip.
2. Extract and move the `addons/healthconnect_plugin` folder into your Godot project's `addons/` directory.
3. In Godot, enable the plugin in **Project > Project Settings > Plugins**.
4. In **Project > Export > Android**, enable the **HealthConnectPlugin** in the Plugins section.

## Demo Project

The `platforms/godot_editor/` directory contains a minimal Godot project that demonstrates all plugin APIs.
